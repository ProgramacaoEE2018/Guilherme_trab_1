# Guilherme_trab_1
Inicialmente, o programa pede para que o usuário insira a quantidade de VE's
feitas e armazena na variável a.
Depois, pede a inserção das respectivas notas, armazenando cada uma delas no vetor ve.
Após isso, a variável soma armazenará o valor da soma de todas as VE's.
Então, a variável mve armazenará a média de VE's, que será dada pela divisão entre soma e a.
Em seguida, pede a inserção da nota de VC (armazenada na variável vc) e associa o valor necessário
para a VF na variável vf.
Por último, verifica se o valor é menor ou igual a 4. Se o for, vf recebe o valor 4.
Finalmente, imprime na tela o valor necessário da VF.
